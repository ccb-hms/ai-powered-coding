# Visual Studio Code Setup for GitHub Copilot

Visual Studio Code (VS Code) provides the most comprehensive and stable integration with GitHub Copilot.

## 1. Install VS Code

If you don't have it installed, download it here:  
[https://code.visualstudio.com/Download](https://code.visualstudio.com/Download)

## 2. Install the Extensions

1. Open VS Code.
2. Click the **Extensions** icon in the left Activity Bar (it looks like 4 squares).
3. Search for and install **GitHub Copilot**.
4. Search for and install **GitHub Copilot Chat**.

## 3. Authenticate

1. Once installed, look at the bottom-right corner of your VS Code window.
2. Click the **Accounts** icon (or the Copilot icon) and select **Sign in with GitHub to use Copilot**.
3. Your web browser will open. Follow the prompts to authorize VS Code.
4. Return to VS Code. You should now see the Copilot Chat icon (a chat bubble) in your left Activity Bar.

## 4. Select Your AI Model

For biomedical data science, we highly recommend using Claude.

1. Open the **Copilot Chat** from the left sidebar.
2. Look for the model dropdown menu (usually at the top of the chat panel, it might say "GPT-4o" or "Auto" by default).
3. Click it and select **Claude 3.5 Sonnet**.
